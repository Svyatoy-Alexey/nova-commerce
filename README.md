# First Web Project

Современный starter на Next.js, TypeScript, Tailwind CSS и App Router.

## Локальный запуск

```bash
pnpm install
pnpm dev
```

Откройте http://localhost:3000. Основная страница находится в `src/app/page.tsx`.

## Проверка

```bash
pnpm lint
pnpm build
```

## Публикация в GitHub

Создайте на GitHub пустой репозиторий без README, затем выполните:

```bash
git remote add origin https://github.com/ВАШЕ-ИМЯ/first-web-project.git
git push -u origin main
```

После публикации репозиторий можно импортировать в Vercel для автоматического развёртывания.
